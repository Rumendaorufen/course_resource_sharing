# 课程资源共享平台项目报告

## 1. 项目概述

### 1.1 项目背景
课程资源共享平台旨在为高校教师和学生提供一个集中的在线教学管理系统。该平台解决了传统教学资源分散、作业管理困难、师生互动不便等问题，为现代化教学提供了有力支持。

### 1.2 项目目标
- 建立统一的课程资源管理平台
- 提供便捷的作业发布与提交系统
- 实现教师与学生之间的高效互动
- 确保教学资源的安全存储和访问
- 提供友好的用户界面和使用体验

## 2. 系统架构

### 2.1 技术架构
#### 前端技术栈
- Vue 3：核心前端框架
- Vue Router：路由管理
- Vuex：状态管理
- Element Plus：UI组件库
- Axios：HTTP客户端

#### 后端技术栈
- Spring Boot 2.7.x：应用框架
- Spring Security：安全框架
- MyBatis Plus：ORM框架
- MySQL 8.0：数据库
- JWT：身份认证
- Maven：项目管理

### 2.2 系统模块
1. 用户认证模块
   - JWT token认证
   - 基于角色的权限控制
   - 安全会话管理

2. 课程管理模块
   - 课程信息管理
   - 教师课程分配
   - 学生选课管理

3. 资源管理模块
   - 文件上传下载
   - 资源分类管理
   - 访问权限控制

4. 作业管理模块
   - 作业发布
   - 在线提交
   - 评分反馈

## 7. 附录

### 7.1 项目结构
```
src/
├── main/
│   ├── java/
│   │   └── com/course/
│   │       ├── config/      # 配置类
│   │       ├── controller/  # 控制器
│   │       ├── service/     # 服务层
│   │       ├── mapper/      # 数据访问层
│   │       ├── entity/      # 实体类
│   │       └── util/        # 工具类
│   └── resources/
└── frontend/
    ├── src/
    │   ├── views/          # 页面组件
    │   ├── components/     # 通用组件
    │   ├── store/          # 状态管理
    │   └── router/         # 路由配置
    └── public/
```
